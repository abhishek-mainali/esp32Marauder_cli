# ESP32 Marauder Web CLI

A premium, feature-rich web interface for the ESP32 Marauder, built with Next.js and Web Serial API.

## Features

- **High-Performance Terminal**: Powered by `xterm.js` for a smooth, lag-free CLI experience.
- **Dynamic Theme System**: Switch between Pink, Green, and Blue "Cyberpunk" themes with a single click.
- **Integrated Command Library**: Interactive guide for all Marauder commands.
- **Signal Visualizer**: Real-time signal monitoring and data visualization.
- **Automated Macros**: Run complex recon and attack sequences with one click.
- **Glassmorphic Design**: Modern, premium UI with responsive sidebars and Zen mode.

## Getting Started

1. **Connect your ESP32 Marauder** via USB.
2. **Open the App** in a supported browser (Chrome, Edge, Opera).
3. **Click "CONNECT"** in the top right corner and select your device's port.
4. **Start Auditing!**
